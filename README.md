## ✨ Ciri-Ciri Utama

- **Satu Fail HTML** — Tiada pemasangan, tiada pelayan, tiada kos
- **localStorage** — Data tersimpan terus dalam pelayar anda
- **Mobile Responsive** — Sesuai untuk telefon, tablet, dan komputer
- **Eksport CSV** — Eksport data ke Excel / Google Sheets
- **Dashboard Masa Nyata** — Statistik dikira secara automatik

## 🚀 Cara Guna

1. Klik pautan GitHub Pages di atas
2. Pilih modul dari tab navigasi
3. Isi borang dan klik **Simpan**
4. Data tersimpan secara automatik dalam pelayar
